# GM-004: Team mode — round-end scoreboard ordering + team totals

**Priority**: P2 | **Status**: TODO

## Problem

From the user's TODO.md: "end of round scoreboard is confusing: it should list players in order of round points. We should also find a way to represent total points for the entire teams."

The round-end scoreboard doesn't clearly show team totals or order players by round performance.

## Affected modes

classic-teams, death-count-teams, long-live-the-king (all team modes)

## Source

User's TODO.md — team mode specific

1. **Client dashboard**: In the round-end summary (from UX-005), order players within each team by round points (descending).
2. **Show team total** prominently in each team's header.
3. **Classic-teams already does this well** (`16_dash_round_end.png`) — use it as the reference implementation and apply the pattern to other team modes.
4. **Tests**: Unit test scoreboard ordering. Test team totals calculation.

## Dependencies

## Estimated scope

Small-medium — the pattern exists in classic-teams. Extend to other team modes.
