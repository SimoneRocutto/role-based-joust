# UX-016: Domination — rework active game UI (phone + dashboard)

**Priority**: P1 | **Status**: TODO | **Tags**: `NEEDS-INPUT`

## Problem

The domination active game UI (both phone and dashboard) looks identical to every other mode. There is no zone indicator, no base status, no capture progress, no team score — none of the domination-specific information is surfaced.

This is listed in the user's TODO.md as "rework domination active game UI".

## Affected modes

domination only

## Audit references

- Domination: `06_phoneA_active.png` — R-P01, R-B03
- Domination: `05_dash_active.png` — R-D04, R-D10, R-D06, R-D02

## Open question (NEEDS-INPUT)

**Phone active screen**: What domination-specific info should be visible?
1. Base/zone status summary (which bases are owned by which team)?
2. Current team scores?
3. Capture progress for nearby base?
4. All of the above?

Constraint: the phone is chest-mounted, so the screen must be readable at a glance. Too much info becomes noise.

**Dashboard active screen**: What layout?
1. Team panels with players + team score (like other team modes) + a base status panel showing all bases
2. A map/grid layout showing bases with team colors and capture state
3. The base status panel already exists (`BaseStatusPanel`) — is it showing during active gameplay?

## Implementation plan (after input)

1. **Client phone**: Add domination-specific info to the active screen
   - Team score and base status could come from existing socket events (`base:status`, `game:tick`)
   - Design a compact layout that fits chest-mounted viewing
2. **Client dashboard**: Ensure `BaseStatusPanel` is visible during active gameplay. Add team scores to team panels.
3. **Tests**: Unit tests for domination-specific phone and dashboard rendering.
4. **Screenshot + audit**: Verify with domination click-through script.

## Dependencies

GM-001 (scoring rework), GM-002 (base capture mechanic), UX-015 (team scores on dashboard)

Large — requires design decisions, new phone UI components, and dashboard layout work specific to domination.
