# UX-023: Dashboard: add HP bars/numbers to player cards

**Priority:** P1
**Status:** TODO
**Tags:** cross-mode
**Audit refs:** R-D04, R-B01

## Problem

Player cards on the active dashboard show HP only via thin border color (green -> yellow -> red). From 3-5 meters this is unreadable. Spectators cannot assess relative HP across players at a glance.

Flagged as CONCERN in: classic-team, role-based, long-live-the-king, death-count, death-count-team, domination (6/7 modes).

## Proposed fix

Add a visible HP indicator inside each player card during active rounds. Options:
- Small HP bar (progress bar) inside the card
- Numeric HP percentage (e.g. "75%")
- Both

The indicator should:
- Be readable at 3-5m projection distance
- Not rely on color alone (bar fill + number)
- Only show during active game state (not lobby/pre-game)

## Files likely affected

- `client/src/components/dashboard/PlayerCard.tsx`
- `client/src/components/dashboard/Scoreboard.tsx` (passes player data)

## Notes

This is the single most frequently flagged CONCERN across the audit. It affects the spectator experience in almost every mode.
